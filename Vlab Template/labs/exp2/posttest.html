Post Test

Q1. In Boolean algebra, which properties are applicable on OR operation?
  (A) Associative properties
  (B) Commutative properties
  (C) Distributive properties
  (D) All of the Mentioned
Ans is (D)


Q2. Identify the type of gate represented by phrase “Any high input guarantees high output”
  (A) OR gate
  (B) AND gate
  (C) XOR gate
  (D) NAND gate
Ans is (A)


Q3.(A + B)(A’.B’) = ?
  (A) 1
  (B) 0
  (C) AB
  (D) AB'
Ans is (B)


Q4.Which gate is simplest, most natural mode of operation for TTL logic
  (A) AND gate
  (B) Inverted AND gate
  (C) Inhibit gate
  (D) NAND gate
Ans is (D)


Q5. Complement of the expression A’B + CD’ is _________
  (A) (A’ + B)(C’ + D)
  (B) (A + B’)(C’ + D)
  (C) (A’ + B)(C’ + D)
  (D) (A + B’)(C + D’)
Ans is (B)


Q6.The logic gate equivalent to negative AND gate is
  (A) OR gate
  (B) AND gate
  (C) NOR gate
  (D) NAND gate
Ans is (C)


Q7.Simplify Y = AB’ + (A’ + B)C.
  (A) AB’ + C
  (B) AB + AC
  (C) A’B + AC’
  (D) AB + A
Ans is (A


Q8.Identify the type of gate represented by phrase “Any low input guarantees low output”
  (A) OR gate
  (B) AND gate
  (C) NOR gate
  (D) NAND gate
Ans is (B)
Q9.NAND gate is equivalent to
  (A) Positive OR gate
  (B) Positive AND gate
  (C) Negative OR gate’
  (D) Negative AND gate
Ans is (C)

Q10.. The Boolean function A + BC is a reduced form of
  (A) AB + BC
  (B) (A + B)(A + C)
  (C) A’B + AB’C
  (D) (A + C)B
Ans is (B)
